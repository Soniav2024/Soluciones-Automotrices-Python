# Soluciones Automotrices - Sistema de Gestión de Reparaciones

## Descripción
Sistema desarrollado en Python para la gestión de reparaciones del taller mecánico "Soluciones Automotrices". Implementa operaciones CRUD (Crear, Leer, Actualizar, Eliminar) sobre un registro de reparaciones.

## Funcionalidades
1. Registrar nueva reparación
2. Listar todas las reparaciones
3. Buscar reparación por ID
4. Actualizar estado de reparación
5. Eliminar reparación
6. Mostrar estadísticas
